# AI Solution Website

This is a **Django-based AI Solutions Platform** that showcases AI solutions across multiple industries. The platform includes:

- Solutions listing and detail pages
- Blog section with list and detail pages
- Articles and Customer Feedback modules
- Contact page and other frontend functionality
- Admin panel for managing solutions, blogs, and content

---
